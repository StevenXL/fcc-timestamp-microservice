API Basejump: Timestamp Microservice

This is a timestamp microservice. The purpose of this microservice is to return a JSON object representing the time passed in as a query string to the service. This JSON object has two properties: unix and natural. (See below for sample inputs).

Example Usage

https://still-spire-9161.herokuapp.com/December%2015,%202015
https://still-spire-9161.herokuapp.com/1450137600

Example Output

{ "unix": 1450137600, "natural": "December 15, 2015" }
